# ccc-findings (`cccf`)

Index Semgrep interrogeable en langage naturel, combiné à [cocoindex-code](https://github.com/cocoindex-io/cocoindex-code) (`ccc`).

Voir `PRD.md` pour le produit et `BACKLOG.md` pour le plan d'implémentation.

## Installation (dev)

```bash
uv sync
uv run cccf version
```

## Tests

```bash
uv run pytest
```
